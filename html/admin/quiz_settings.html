Supported Formats:
Option A (Q-A format):
- Separate question blocks with double spacing (blank lines).
- Question starts with optional Q1., Q2., etc.
- Options start with A., B., C., D. (or with parenthesis like A)).
- The correct answer is indicated by a line starting with Answer: A (or Ans: A).
Example:
Q1. What does HTML stand for?
A. Hyper Text Markup Language
B. High Text Machine Language
Answer: A Option B (Asterisk format):
- First line is the question.
- Options on following lines.
- Start the correct option with an asterisk (*).
Example:
What is 2 + 2?
3
*4
5
